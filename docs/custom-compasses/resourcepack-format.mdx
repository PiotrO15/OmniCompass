# Custom Compass Resource Pack

The model and the name of the compass can be customized using a resource pack.

## Compass Model

To make compass textures easier to change, OmniCompass model loader uses two separate models located in `assets/<namespace>/models/compass/`: `<compass_name>.json` and `<compass_name>_needle.json`. This means you only need to create a single texture for the compass if you don't need to change the look of the needle. 

Default biome compass' base model:
```json
{
  "parent": "minecraft:item/generated",
  "textures": {
    "layer0": "omnicompass:item/biome_compass_base"
  }
}
```

## Needle Colors

You can apply a color tint to the compass's needle and its shadow using a [datapack](./datapack-format#needle-colors).

## Compass Translation

Each compass type requires a translation for its name, which has to be specified in `assets/<namespace>/lang/en_us.json`:

```json
"compass_type.<namespace>.<compass_name>": "Compass Name"
```
